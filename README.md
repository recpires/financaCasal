## 📌 Sobre o Projeto

Este projeto é a materialização do front-end de uma aplicação Fintech focada em **contas conjuntas residenciais**. O objetivo é oferecer uma interface moderna e intuitiva onde o casal pode acompanhar, em um único painel, suas receitas, despesas, saldo consolidado e metas financeiras.

A interface foi projetada utilizando o conceito de **Dark Mode**, complementada por efeitos de *Glassmorphism* (vidro fosco) e detalhes em azul neon, garantindo um aspecto premium e de alta legibilidade.

*Projeto desenvolvido como requisito prático e avaliativo para o curso da FIAP.*

## ✨ Funcionalidades e UI/UX

* **Design Responsivo:** Layout fluido que se adapta perfeitamente a dispositivos móveis (Mobile First) e desktops, utilizando o sistema de grid e flexbox do Tailwind.
* **Glassmorphism:** Utilização de transparências e desfoques (blur) nos cards para criar uma hierarquia visual sofisticada.
* **Dark Mode Nativo:** Fundo escuro texturizado com alto contraste nos dados financeiros (Entradas em esmeralda, Saídas em rosa).
* **Separação de Preocupações:** Código HTML semântico isolado da estilização CSS customizada, garantindo boas práticas de desenvolvimento.

## 🛠️ Tecnologias Utilizadas

A construção desta interface utilizou uma stack leve e focada em produtividade e design:

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilizações complementares e customização de scrollbars.
* **Tailwind CSS (via CDN):** Framework utility-first utilizado para toda a componentização e responsividade da interface.
* **FontAwesome:** Biblioteca de ícones vetoriais para representação visual das categorias financeiras.
* **Google Fonts:** Tipografia 'Inter' para garantir uma leitura clara e moderna dos dados.

## 🚀 Como Executar o Projeto

Como o projeto foca na estruturação Front-end utilizando Tailwind via CDN, a execução é extremamente simples e não requer nenhum ambiente de compilação ou Node.js.

1.  Faça o clone deste repositório:
    ```bash
    git clone https://github.com/recpires/financaCasal.git
    ```
2.  Acesse o diretório do projeto:
    ```bash
    cd financaCasal
    ```
3.  Abra o arquivo `index.html` diretamente em qualquer navegador moderno (Google Chrome, Firefox, Edge, etc.).
